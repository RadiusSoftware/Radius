Server Setup

Welcome to the Radius Software web application client-server framework. You will be able to set your host up as a standalone application server or attach your host to an existing application swarm. We will guide you through this process step-by-step, after which an encrypted file called "boot" will be created and placed in the radius software parent directory. Here's an overview of the following process:
  • Ensure you host is associated with a registered Internet domain: A or AAAA record.
  • Contact your ACME certificate authority to obtain TLS and CAA certificates and then restart in TLS mode.
  • Decide whether to configure a standalone or a swarm server.

Standalone Mode

  • Create and configure a Mailgun account
  • Enter your Mailgun account credentials
  • Enter a valid email address
  • Verify the entered email address
  • Set your user password
  • Save your boot file
  • Record you startup key

Swarm Mode

  • Enter a Swarm ID
  • Enter a Swarm Host
  • Save your boot file
  • Record you startup key
